# Spec

Dette bibliotek samler projektets specifikationer og beslutninger.

## Filer
- `domain.md` - Domænebegreber, regler og afgrænsning
- `use-cases.md` - Aktører, scenarier og acceptkriterier
- `GENERATION.md` - Regler for AI-assisteret kodegenerering/ændringer
- `decisions.md` - Arkitekturbeslutninger (ADR-light)

## Arbejdsmåde
1. Start i `domain.md`.
2. Beskriv flows i `use-cases.md`.
3. Registrér beslutninger i `decisions.md`.
4. Følg reglerne i `GENERATION.md` ved kodning.
